# LLM Wiki Starter Skill v1.6.2

This Skill creates or updates an Obsidian-ready Japanese LLM Wiki.

It is intentionally no-script: the agent creates the directory tree directly from the user's purpose and hearing selections.

## Key features

- Purpose-specific directories under `10_素材/`, `20_Wiki/`, and `30_成果物/`
- Selected Wiki items become real directories under `20_Wiki/`
- Selected Outputs items become real directories under `30_成果物/`
- HOME default sections are always included
- `index.md` and `log.md` are mandatory
- `個人用メモ/` is excluded from wiki processing and git
- Privacy and sensitive-information hearing is required
- Standard privacy mode keeps human names as written, while redacting personal contact information, HR/personnel information, contract amounts by default, passwords, and production credentials
- Strict mode redacts names as well
- Custom mode lets the user choose handling per category
- The selected privacy policy is reflected in `個人情報・機密情報ポリシー.md`, `AGENTS.md`, `CLAUDE.md`, `HOME.md`, `README.md`, `log.md`, and `wiki設定.json`
- Passwords, API keys, tokens, SSH keys, cookies, and production credentials are never copied into generated wiki pages or outputs

## Upload note

The files inside this Skill package use ASCII-only paths to avoid upload errors caused by non-ASCII ZIP paths.

The Wiki generated by this Skill may still use Japanese directory and file names.
