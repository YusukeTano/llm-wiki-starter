# LLM Wiki Starter Skill v1.9.1

Obsidianで運用しやすい日本語中心の LLM Wiki を作成・更新するSkillです。

## v1.9.1 の主な変更

- README.mdの `このWikiの使い方プロンプト集` を、固定の汎用文ではなく、Wikiの目的・実ディレクトリ・Outputs・HOME項目・個人情報ポリシーに合わせて生成するようにしました。
- MSP/Slack Q&AならSlack Q&A用、研究なら論点・仮説用、読書なら章別要約用、旅行なら旅程・予約用のプロンプトになります。
- 4つの基本ワークフローに加えて、Wikiに質問する、回答を保存する、健康診断をする、個人情報・機密情報を点検するプロンプトも目的別に生成します。
- ディレクトリ作成スクリプトは含みません。LLMが目的とヒアリング結果に沿って構成を設計します。

## Skill内部のファイル

```text
SKILL.md
README.md
manifest.txt
```

Skillパッケージ内部はASCII名だけにしています。生成されるWiki側では日本語ディレクトリ名・日本語ファイル名を使えます。
